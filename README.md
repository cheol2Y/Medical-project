## 📕 빅데이터 기반 의료 상담 추천 서비스 (가제)
### 📗 증상을 채팅으로 상담해 병(결과)을 도출하고 결과에 따른 다양한 서비스 운용.
### 📗 위치 기반 병원, 약국 추천 / 약 성분 조회 등.
### 📗 증상 별 병원 메이저 선택. 
### 📗 예상 진료비 예측, 계산.
### 📗 실시간 상담 내용 중 가장 많은 증상, 병명 등을 조회해서 실시간 랭킹.
